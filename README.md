# ToDo App

A simple ToDo app created for the academic course with aim of creating an app with frontend, backend and database.

## Technology Stack

1. Frontend
*	Angular 16.2.0,
*	TypeScript 5.1.3,
*	Angular Material 16.2.8,
*	Scss 0.2.4,
*	Rxjs 7.8.0,
*	Html 5.

2. Backend
*	Java 17,
*	Spring Boot 3.14,
*	Spring Data JPA,
*	Gradle Groovy,
*	PostgreSQL,
*	MapStruct,
*	Lombok.

3. Others
*	Docker,
*	Nginx,
*	Drawio.
